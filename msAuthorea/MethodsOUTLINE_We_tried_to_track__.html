Methods

OUTLINE:
We tried to track down any data that were collected based on funding from the EVOSTC, which requires making public any data 1 year after collection.
Steps to data collection and publication:
  1. Started with a list of projects: PI names and project numbers (some included project descriptions and most, years funded and associated agency)
  2. Searched for contact info of project PIs (google, agency sites)
  3. Where contact info was located, outreach emails were sent explaining the data recovery and archiving project and requesting data for the funded project(s)
    1. when possible, calls followed unreplied emails
  4. A reply to the email (or call) signified that we had accurate contact information and that the researcher was aware of our request for the data
    1. Replies gave us an indication of willingness to cooperate: “yes, I’ll get you that”, to “where do you get the right to ask me for data?!”
  5. once (if) data were sent, we worked with the data owners/custodians to format the data in persistent formats and create EML metadata
  6. Once researcher approves the data package, publish to public online database

At the end of this 2 year effort, datasets are characterized as….
  • Email sent: find contact information and reach out for the data from a specific EVOSTC project
    • email (multiples)
      • PI and other associated parties
    • call
    • find other sources of these data within agency
  • replied: we have the correct contact information and data request has been received
    • potentially discussed data request and work it would take to obtain it
    • potentially multiple contacts (eg: talk to this person about getting those data…)
  • sentData: data were received from the owner/contact
    • contact has sent the data and/or metadata
    • may need to continue to work with the scientist of contact to prepare the data/metadata for publication
  • dataRevised: data have been successfully formatted and metadata created
    • Data can be stalled here if the researcher was cooperative in providing data and assisting with formatting but requested for the data to not be published until a later date
    • Alternatively temporary revised status exists before the PI is able to review the completed package and accept for publication
  • published: data &/or metadata have been published to the GOA data portal
  • Unrecoverable: various reasons data were determined lost or otherwise unavailable

Statistical Analysis:

Differences between groups was assessed using simple binomial statistical tests: Chi-squared and binomial linear regression (BLM). If the chi-squared test showed differences between groups, a BLM was run to identify which groups resulted in the biggest positive or negative result.
